The Kolkata Smart City Dashboard is an interactive Power BI project developed to analyze urban infrastructure and smart city performance metrics using modern data visualization techniques. This dashboard focuses on presenting meaningful insights related to city development, infrastructure trends, and urban analytics through an intuitive and user-friendly interface.
The project was created using Power BI, DAX, Power Query, and Excel for data transformation, modeling, and visualization. Various analytical components such as  geographic visualizations, trend analysis, slicers, and category-based filtering were implemented to improve data exploration and storytelling. The dashboard helps users understand urban performance indicators and compare city metrics effectively through interactive reports.
The project demonstrates practical skills in data cleaning, dashboard design, business intelligence reporting, and analytical thinking. Special emphasis was placed on creating a modern and visually appealing dashboard layout while maintaining readability and usability. DAX measures and data modeling techniques were used to generate dynamic insights and improve analytical capabilities.
